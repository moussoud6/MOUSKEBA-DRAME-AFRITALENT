#Afritalent
projet fil rouge - plateforme de mise en relation entre freelances africains et clients.
Auteur:MOUSKEBA DRAME
promotion:L1 web - ISI
## description
Afritalent est un site vitrine fictif connectant les meilleurs freelances tech africains avec des entreprises ambitieuses. Il presente les services, les tarifs, les profils de freelances et permet aux visiteurs de contacter la plateforme.
### technologies utilisees
- HTML
- CSS3
- Bootstrap
- JavaScript
- Git & Github
#### fonctionnalites principales
- Dark mode avec localStorage
- Compteurs animes au scroll
- Filtrage dynamique des freelances
- Validation du formulaire de contact
- Navbar dynamique au scroll
- Bouton retour en haut
- Animations fade-in au scroll
